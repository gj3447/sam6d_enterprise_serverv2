# Archive

이 폴더에는 과거 파일들이 아카이브되어 있습니다.

## 📂 구조

### `docs_old/`
이전 문서 파일들 (25개)
- 과거 구현 상태 문서
- 테스트 결과 문서
- 문제 해결 가이드 등

### `test_history/`
과거 테스트 스크립트 (30개)
- 개발 과정에서 생성된 테스트 파일들
- 디버깅용 스크립트
- 초기 테스트 코드

### `tests_backup/`
중복된 테스트 파일 (7개)
- 유사한 기능의 테스트 파일들
- 더 이상 사용하지 않는 스크립트

## ⚠️ 주의사항

- 이 폴더의 파일들은 더 이상 사용되지 않습니다
- 참조용으로만 보관되어 있습니다
- 필요시 복원은 가능하지만 권장하지 않습니다

## 🔍 파일 찾기

특정 파일을 찾으려면:
```bash
# PowerShell
Get-ChildItem -Recurse -Filter "*.py" archive/

# Linux/Mac
find archive/ -name "*.py"
```

## 📝 복원 방법

필요한 파일이 있다면:
1. 파일 위치 확인
2. `archive/`에서 원하는 위치로 복사
3. 사용 후 다시 정리
